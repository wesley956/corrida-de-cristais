# Fragment Rush: Corrida dos Cristais
## Guia Visual v0.2 — Cultivation / Wuxia

Esta atualização transforma a base visual do jogo em uma experiência de corrida espiritual, misturando cristais, energia celestial, fluxo, ressonância e ascensão.

## Direção oficial

**Rápido no controle, suave na sensação e espiritual na apresentação.**

O jogo continua sendo um runner mobile de coleta e desvio, mas agora com atmosfera de cultivation/wuxia: névoa, jade, céu profundo, trilhas espirituais, montanhas flutuantes, núcleo errante e efeitos de ressonância.

## Termos aplicados no código

| Antes | Agora |
|---|---|
| Dash | Passo Espiritual |
| Rasante Perfeito | Ressonância Perfeita |
| Combo | Fluxo |
| Recorde | Marca de Ascensão |
| Loja | Pavilhão das Formas |
| Player inicial | Núcleo Errante |

## Mudanças visuais aplicadas

- Fundo azul profundo com camadas espirituais.
- Lua/glow celestial no menu e gameplay.
- Montanhas flutuantes com parallax.
- Partículas de qi/jade no fundo.
- Faixas espirituais no caminho.
- Cristais com brilho e rotação.
- Obstáculos renomeados: Ruptura Celestial, Espinho de Cristal, Selo Instável e Fragmento Caído.
- Player com aura, halos, anéis e pulsação.
- UI mais suave, translúcida e temática.
- Tela de resultado como “Fluxo Interrompido”.
