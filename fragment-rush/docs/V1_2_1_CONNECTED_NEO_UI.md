# Fragment Rush v1.2.1 — Connected Neo UI

## Correção importante
A v1.2 adicionou as cenas novas, mas o Main.gd ainda mostrava a UI antiga.
Esta versão conecta o FragmentUiController ao Main.gd.

## Agora conectado
- Menu usa NeoMenuScreen
- Pavilhão usa NeoPavilionScreen
- Núcleo usa NeoCoreScreen
- Botões das cenas chamam as funções reais do jogo
- Dados reais alimentam as cenas: XP, círculos, cristais, formas, técnicas
- UI antiga de Menu/Pavilhão/Núcleo fica escondida

## Observação
Resultado, Tutorial, HUD e jogo em si ainda usam a base antiga por enquanto.
