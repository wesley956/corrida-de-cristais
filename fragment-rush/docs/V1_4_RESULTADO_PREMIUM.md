# Fragment Rush v1.4 — Resultado Premium + Recompensas Animadas

## Objetivo
Transformar a tela de fim de corrida em uma tela de recompensa, não um relatório.

## Novidades
- Contagem animada de XP e Cristais.
- Badges visuais para XP, Cristais e Bioma.
- Missões concluídas aparecem como cápsulas visuais.
- Tela de resultado com mais brilho, arcos e sensação premium.
- Botão direto para Pavilhão.
- Botão direto para Núcleo.
- Layout do resultado reorganizado para reforçar “perdi, mas evoluí”.
