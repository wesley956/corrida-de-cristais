# Fragment Rush v1.3 — Formas + Rastros Premium

## Objetivo
Transformar as formas em recompensas mais desejáveis, com diferença visual real durante a corrida.

## Novidades
- Cada forma agora tem variação de silhueta.
- Cada forma tem cor secundária própria.
- Cada forma tem rastro próprio:
  - Núcleo Errante: Rastro Cristalino
  - Semente de Jade: Rastro de Jade
  - Orbe Celestial: Rastro Celestial
  - Coração Nebular: Rastro Nebular
  - Essência Dourada: Rastro Dourado
- Raridade aumenta presença visual de aura.
- Player muda de formato conforme a forma equipada.
- Preview do Pavilhão ganha desenho mais distinto por forma.
- Overlay de desbloqueio: “Nova Forma Desperta”.
- Tela do Pavilhão mostra o rastro/afinidade da forma.
